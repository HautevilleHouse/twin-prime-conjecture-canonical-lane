# Identification Bridge

Gate: `TP_G6`
Constant: `eps_coh`

The determining class `C_det` is used to compare extracted endpoint observables with the intended target observables. The bridge closes when

`eps_coh = 0`

in strict mode, so there is no residual discrepancy between the lane endpoint and the target statement.

In mainstream terms, this is the no-hidden-gap identification step.
